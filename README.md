# Risk management tool kit
## 前言
在準備FRM（金融風險管理師）考試的過程中，我學習到了許多關於風險管理的理論與實踐方法。這些方法讓我對市場風險、信用風險、操作風險等有了更深入的理解。為了將所學知識應用於現實，
我計劃撰寫Python代碼，打造一個集成多種風險管理功能的工具箱。這個工具箱將能夠幫助我更高效地分析市場數據、評估風險敞口、進行情境分析，並且可以在日常的風險管理工作中靈活運用，
實現學術與實務的結合。
